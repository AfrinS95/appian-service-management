# appian-service-management
an enterprise-style service request platform where employees can submit IT, HR, Access, Procurement, and Facilities requests. The system uses AI to classify requests, suggest priority, generate ticket summaries, recommend knowledge articles, and assist agents with response drafting.

# Intelligent Service Request Management Platform

## Overview

An enterprise-grade Appian application that manages:

- IT Requests
- HR Requests
- Procurement Requests
- Facility Requests

The platform uses Appian AI capabilities for:

- Request Classification
- Priority Prediction
- Ticket Summarization
- Knowledge Article Recommendations
